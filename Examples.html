(static) webexDevice camera Control

webex Device Macro camera Control example.

  • Click on source to see the script example.

File Discribtion

  • jibb.js -> entry point.
  • jibbWebexXapi.js -> jibb SDK responsbile HTPP requests.

Installation

  • downalod/get jibb.js.
  • Get jibbWebexXapi.js from "www.s3.*******.com" for latest jibb sdk or you could use the file in webexDevice jssdk folder (could be out of date).
  • Log into your Cisco Room Device as and admin.
  • Navigate to the Macro Editor.
  • Import and Save each of the following Macros into the Room System: jibbWebexXapi.js and jibb.js .
  • Edite jibb.js by adding camera postion to sanp to when clicked in line 114 or comment line 153 to disable it.
  • Edite jibb.js by adding your personal Jibb Api Key to in line 182, to get your Api key visit https://app.jibb.ai/ then navigate to personal settings and click Generate.
  • Save and activate jibb.js .
  • After refreshing the webage you will find jibb panel automatically added in the UI Extension Editor. variable uiExtension line 242 is is the exported XML panel
